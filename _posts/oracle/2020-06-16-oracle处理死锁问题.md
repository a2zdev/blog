检查oracle是不是存在锁：

```sql
select l.session_id,
       s.serial#,
       l.locked_mode,
       l.oracle_username,
       s.user#,
       l.os_user_name,
       s.machine,
       s.terminal,
       a.sql_text,
       a.action
  from v$sqlarea a, v$session s, v$locked_object l
 where l.session_id = s.sid
   and s.prev_sql_addr = a.address
 order by sid, s.serial#;
```

杀掉对应的会话：

```sql
alter system kill session '2279,97'
```

2279为session_id
97为serial#
